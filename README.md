# TimeTrack QR

Smart Attendance System based on QR Code.

---

## Author

**Bonte Tabala**  
Founder & Lead Software Architect

---

## Description

TimeTrack QR is a professional attendance management platform designed for companies, institutions and schools.

The system allows:

- Secure attendance tracking with QR Code
- Employee presence monitoring
- Real-time attendance validation
- Administrative dashboard
- Attendance history and reports

---

## Technologies

Backend:
- AdonisJS
- PostgreSQL
- Lucid ORM
- VineJS validation
- Google OAuth

Frontend:

- TypeScript
- Tailwind CSS
